# Mon Vieux Grimoire - Backend

## Installation

1. Cloner le projet
2. Se placer dans le dossier backend :
   cd backend

3. Installer les dépendances :
   npm install

4. Créer un fichier .env basé sur le fichier .env.example

5. Renseigner vos propres variables :
   - MONGO_URI
   - JWT_SECRET
   - PORT

6. Lancer le serveur :
   node server.js

Le serveur démarre sur http://localhost:4000
